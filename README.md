Proyecto Intermodular ASIR – Contact Center

Descripción

Este proyecto consiste en el diseño completo de la arquitectura tecnológica de un Contact Center con 55 usuarios, integrando todos los módulos del ciclo ASIR:

- Redes (VLANs, ACLs, DHCP, WiFi)
- Sistemas Operativos (Windows Server, Linux)
- Bases de Datos (MySQL)
- Lenguajes de Marcas (XML, XSD, XPath)
- Hardware (dimensionado por perfiles)
- Cloud (AWS)

---

Arquitectura

El sistema está diseñado en varias capas:

1. **Hardware** → Equipos por perfil (agentes, técnicos, admin)
2. **Red** → Segmentación mediante VLANs
3. **Sistemas** → Active Directory y control de accesos
4. **Base de datos** → Modelo relacional de llamadas
5. **Datos (XML)** → Validación y estructuración
6. **Cloud** → Escalabilidad en AWS

---

Objetivo

Simular una infraestructura real donde:

- Los usuarios trabajan segmentados por roles
- Los datos se gestionan de forma estructurada
- La red garantiza seguridad y control de acceso
- El sistema puede escalar a entornos cloud

---

Tecnologías utilizadas

- Cisco Packet Tracer  
- MySQL / Workbench  
- XML / XSD / XPath  
- Windows Server / Linux  
- AWS (EC2, RDS, S3)

---

Estructura del proyecto

El proyecto está organizado por módulos del ciclo ASIR:

- 0370 – Planificación y Administración de Redes  
- 0371 – Fundamentos de Hardware  
- 0369 – Implantación de Sistemas Operativos  
- 0372 – Gestión de Bases de Datos  
- 0373 – Lenguajes de Marcas  
- MPO – Computación en la Nube
